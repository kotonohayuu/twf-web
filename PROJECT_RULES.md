# TWF-Web プロジェクトルール

## 正本（マスター）の定義

- **GitHubリポジトリ（`twf-web`）を正本とする。**
- `note.md` の最新版は常にGitHub上のものが正とする。
- Notion等の他の場所にコピーが存在する場合でも、GitHub版が優先される。

## 画像の管理

- 画像はすべて `./images/` ディレクトリに格納する。
- 画像形式は **WebP** を優先する（`.webp`）。
- `note.md` 内の画像参照は `![altテキスト](./images/ファイル名.webp)` の形式を使用する。
- altテキストにはキャラクター名と簡潔な説明を含める。

## デプロイ

- GitHubへのpushにより、Vercel（`twf-web.vercel.app`）に自動デプロイされる。
- デプロイ反映には約50秒かかる。

## 編集フロー

1. `note.md` または画像を編集
2. `git add` → `git commit` → `git push`
3. Vercelデプロイの自動反映を確認

## 備考

- このリポジトリはプライベート設定。
- Vercel上のサイトが公開閲覧用のフロントエンド。
